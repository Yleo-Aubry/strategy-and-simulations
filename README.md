Welcome to my experimental repository. Here, I apply **Data Science, Game Theory, and Algorithms** to competitive systems (Sports, Board Games, etc.).

##  Projects

### [UCL Football Predictor](./ucl-football-predictor)
A Monte Carlo simulation engine for the UEFA Champions League 2026.
* **Core:** Python, NumPy, Pandas.
* **Maths:** Elo Ratings, Dixon-Coles adjustment, Poisson distribution.
* **Output:** 10,000 simulations to determine qualification probabilities.
